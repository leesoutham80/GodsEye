GodsHand Delta

GodsHand is the machine\u2019s independent assessment of the same crisis using the same signal data but without Lee\u2019s interpretive bias.

How it works: The raw signal values are sent to Claude with a system prompt that instructs independent analysis. The machine may disagree on signal weights, mode selection, exit timing, or scenario probabilities.

The Delta \u2014 where Lee and the machine disagree \u2014 is the product. Agreement = high confidence. Divergence = requires investigation.

Colour coding: ● Green = agreement (±20). ● Yellow = minor divergence (±40). ● Red = major divergence (±60+).

GodsEye Tier 1 (\u00a3500/mo) = Lee\u2019s view. GodsHand Tier 2 (\u00a31,500/mo) = Lee + Machine. The delta between them is where edge lives.
